ðŸ”¹ 1. Block vs Inline Elements

  • Block-level elements: Occupy full width, start on a new line.
  • Inline elements: Take only as much width as needed, do not start on a new line.

Examples of Block Elements:

  • <div>, <p>, <section>, <article>, <table>, <form>, <header>, <footer>

Examples of Inline Elements:

  • <span>, <a>, <strong>, <em>, <img>, <mark>, <sub>, <sup>

ðŸ”¹ 2. ID vs Class

ID [#]

  • Used to uniquely identify an element.
  • No two elements should have the same ID.
  • One element can have only one ID.
  • To select an ID in CSS: #idName
  • Invalid: id="id1 id2" â€“ multiple IDs not allowed
  • Example: <p id="ptag">Paragraph</p>

Class [.]

  • Used to apply styles to multiple elements.
  • Multiple elements can share the same class.
  • One element can have multiple classes.
  • To select a class in CSS: .className
  • Valid: class="btn primary highlight"
  • Example: <p class="ptag">Paragraph</p>
---

ðŸ”¹ 3. Formatting Tags (Inline)

  • <strong>: Bold text (semantic)
  • <em>: Italic text (semantic)
  • <mark>: Highlighted text
  • <ins>: Inserted/underlined text
  • <sub>: Subscript (e.g., H2O)
  • <sup>: Superscript (e.g., x2)
