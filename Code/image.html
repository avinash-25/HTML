ðŸ”¹ Image Tag Notes

  • <img> tag is used to display images on a webpage.
  • This is a self-closing and inline tag (does not need a closing tag).

    • ðŸ”¸ Syntax:

    <img src="image.jpg" alt="Description" width="300" height="200">

    ðŸ”¸ Important Attributes:

    • src: (source) URL or file path of the image (required).
    • alt: (alternate text) describes the image if it doesn't load or for screen readers (required for accessibility).
    • width / height: Defines the size of the image (in px or %).
    • title: Tooltip text shown on hover.
    • loading: Accepts "lazy" to delay loading until the image is in view (improves performance).

    ðŸ”¸ Types of Image Sources:

    • Relative Path: Image is in the same project folder.
      Example: src="images/pic.jpg"
    • Absolute Path: Full URL of the image.
      Example: src="https://example.com/logo.png"

    ðŸ”¸ Example 1: Basic Image

    <img src="pic.jpg" alt="Nature image" width="300" height="200">

    ðŸ”¸ Example 2: With Tooltip

    <img src="pic.jpg" alt="Lake view" title="Hover text - Lake view">

    ðŸ”¸ Example 3: Lazy Loading

    <img src="pic.jpg" alt="Lazy load image" loading="lazy">

    ðŸ”¸ Notes:

    • Always use alt for better SEO and accessibility.
    • Use compressed images for faster loading.
    • Prefer relative paths for local projects.
    • Use loading="lazy" to improve webpage performance.
