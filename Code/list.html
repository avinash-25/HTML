Class Notes

  • List: A list is a structured way to group related items in HTML.
  • Types of Lists in HTML:
    • Ordered List (<ol>): Displays items in a specific sequence (numbered or lettered).
    • Unordered List (<ul>): Displays items without order (bullets).
    • Description List (<dl>): Used for definitions (term + description).

    Ordered List (<ol>) Details:

    • Uses numbers or letters to order items.
    • Common type attribute values:
      • 1 – Default (1, 2, 3, ...)
      • A – Uppercase letters (A, B, C...)
      • a – Lowercase letters (a, b, c...)
      • I – Upper Roman (I, II, III...)
      • i – Lower Roman (i, ii, iii...)

    Unordered List (<ul>) Details:

    • Uses symbols like bullets.
    • Common type attribute values:
      • disc – Default filled circle
      • circle – Hollow circle
      • square – Square box
      • none – No bullet

    Description List (<dl>) Details:

    • <dl> – Description List container
    • <dt> – Term (the item being defined)
    • <dd> – Description (the definition or explanation)
    • Used for FAQs, glossaries, key-value pairs

    Example of Ordered List:

    <ol type="A">
    <li>Pen</li>
    <li>Pencil</li>
</ol>

    Example of Unordered List:

    <ul type="square">
    <li>Milk</li>
    <li>Bread</li>
</ul>

    Example of Description List:

    <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>

    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>
  • Lists make webpages more organized and readable.
