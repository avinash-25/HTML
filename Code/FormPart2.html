ðŸ”¸ Label Tag

  • The <label> tag is used to define a label for an input element.
  • It helps improve form accessibility and connects text with the input field.
  • The for attribute of <label> must match the id of the input field.

ðŸ”¸ Additional Notes

  • Server has CPU and memory to store and process form data.
  • Client and server communicate using URLs.
  • Form data is sent to the server using the name attribute of each input field.
  • When using method="get", the data is sent via URL with ? used to separate values.
  • To keep sensitive data secure (like passwords), use method="post".
  • Value vs Placeholder:
    • Value shows permanent default content in the input.
    • Placeholder gives a hint which disappears on typing.
